* Fur Elise: 
	- 135 BPM
	- 3/8 time

Duration of quarter note = (1/135)\*60\*60 = 26.6666 ms
So:
	106.6666 ms - Whole
	53.3333 ms  - Half
	26.6666 ms  - Quarter
	13.3333 ms  - Eigth
	6.6666 ms   - Sixteenth
